# 空頭趨勢策略 v2

> **商品**：加密貨幣（交易對可設定）
> **週期**：可設定（預設 5 分鐘 K 線）
> **方向**：做空（Short）

---

## 一、指標定義

| 指標 | 說明 |
|------|------|
| **HA_Close** | 平均K線收盤價 = (Open + High + Low + Close) / 4 |
| **HA_Open** | 平均K線開盤價 = (前一根 HA_Open + 前一根 HA_Close) / 2 |
| **HA_WMA2** | 以 HA_Close 為來源的加權移動平均線，週期 = 2 |
| **HA_WMA4** | 以 HA_Close 為來源的加權移動平均線，週期 = 4 |
| **ATR** | 原始K線的平均真實波幅，週期與倍數從 config 設定 |

---

## 二、建倉條件（Entry）

必須**同時滿足**以下兩個條件，才在下一根 K 線開盤時掛限價空單：

### 條件 1 — 趨勢過濾（死亡交叉）

- **Bar[0]** 收盤後：`HA_WMA2[0] < HA_WMA4[0]`
- **Bar[-1]**（前一根）：`HA_WMA2[-1] >= HA_WMA4[-1]`
- 即 HA_WMA2 由上往下穿越 HA_WMA4，形成**死亡交叉**

### 條件 2 — 趨勢結構確認

- `HA_Close[-2] > HA_Close[0]`
- `HA_Close[-3] > HA_Close[0]`
- 即交叉前第 2、第 3 根的平均K收盤價皆高於交叉當根，確認趨勢在交叉前已持續下跌

### 建倉動作

- 在 **Bar[1]** 開盤時，以**限價單**開空單
- 限價價格 = 真實K線 Bar[1] 開盤價 - 滑點（config 設定）
- 滑點方向往下，確保在開盤瞬間能快速成交

```
時間軸示意：

... | Bar[-3] | Bar[-2] | Bar[-1] | Bar[0] 死叉發生（收盤後判斷）| Bar[1] 進場 |
       ↑           ↑                        ↑                          ↑
  HA_Close[-3]  HA_Close[-2]           條件1判斷                  限價賣出
  需 > HA_Close[0]  需 > HA_Close[0]                        = 真實開盤 - 滑點
```

---

## 三、止損策略（Stop Loss）

使用原始K線的 **ATR 移動止損軌道**，止損只能往有利方向（向下）移動。

### 止損計算

```
止損價 = 近 N 根原始K線最低價 + ATR(period) × multiplier
```

- `N`、`period`、`multiplier` 皆從 config 設定
- 每根 K 線收盤後重新計算

### 移動規則

```python
new_stop = lowest_low(N) + ATR(period) * multiplier

if new_stop < current_stop:
    current_stop = new_stop  # 只能往下調，不可倒退
```

- 止損只能下移，確保保護住的獲利不會因 ATR 擴大而縮水

---

## 四、止盈策略（Take Profit）

目前以**移動止損帶出**為主，無固定止盈點：

- 當原始K線價格突破 `current_stop` 時，以市價平倉
- 讓獲利自然奔跑，由市場決定出場時機

---

## 五、Config 參數

| 參數 | 說明 | 建議初始值 |
|------|------|-----------|
| `wma_fast` | 快線週期 | 2 |
| `wma_slow` | 慢線週期 | 4 |
| `atr_period` | ATR 計算週期 | 14 |
| `atr_multiplier` | ATR 止損倍數 | 2.0 |
| `atr_lookback` | 計算最低價的回看根數 | 14 |
| `slippage` | 限價單滑點（以報價單位計） | 依交易所調整 |

## 六、邏輯流程圖

```mermaid
flowchart TD
    A[開始監控] --> B{Bar[0] 收盤\nHA_WMA2 下穿 HA_WMA4？}
    B -- 否 --> A
    B -- 是 --> C{HA_Close[-2] > HA_Close[0]\n且 HA_Close[-3] > HA_Close[0]？}
    C -- 否 --> A
    C -- 是 --> D[Bar[1] 開盤\n掛限價空單\n= 真實開盤 - 滑點]
    D --> E[初始止損\n= 近N根最低價 + ATR × 倍數]
    E --> F{持倉中}
    F --> G[每根K收盤後\n重新計算止損]
    G --> H{new_stop < current_stop？}
    H -- 是 --> I[下移止損至 new_stop]
    H -- 否 --> J{價格突破 current_stop？}
    I --> J
    J -- 否 --> F
    J -- 是 --> K[市價平倉出場]
    K --> A
```
